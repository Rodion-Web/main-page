# Quattr Group Landing Page

Профессиональный адаптивный landing page для презентации инвестиционного продукта в сфере недвижимости. Проект выполнен на чистом HTML, CSS и JavaScript без обязательного build-процесса и может быть запущен как статическая страница.

## О проекте

Этот репозиторий содержит многоэкранный промо-сайт с акцентом на:

- презентацию услуги комплексного сопровождения инвестора;
- визуально выразительную подачу продукта;
- адаптивное отображение на desktop и mobile;
- понятные пользовательские сценарии с CTA-блоками и формой обратной связи.

Интерфейс построен как статический frontend-проект с кастомной стилизацией, собственным JavaScript-компонентом карусели и набором секций для демонстрации оффера, преимуществ и объектов.

## Основные возможности

- Hero-слайдер с автопрокруткой и dot-навигацией.
- Кастомный carousel для карточек объектов с кнопками `prev/next`.
- Поддержка touch-свайпа и клавиатурной навигации в слайдерах.
- Адаптивные блоки для мобильной и десктопной версии.
- Форма консультации с выпадающим списком международных телефонных кодов.
- Использование `aria-label`, фокус-стилей и семантических секций для базовой доступности.

## Технологии

- `HTML5`
- `CSS3`
- `Vanilla JavaScript`
- локальные шрифты `Futura New`

## Архитектура frontend-части

### `index.html`

Содержит основную разметку лендинга: hero-блок, секции с преимуществами, блоки "Как это работает", карточки объектов и форму консультации.

### `css/main-page.css`

Стили организованы вокруг design tokens в `:root`, адаптивных breakpoint-переменных и отдельных блоков интерфейса. В проекте используется единая цветовая система и собственная типографика.

### `js/main-page.js`

Ключевая логика сосредоточена в двух частях:

- класс `Carousel` для fade- и track-слайдеров;
- инициализация dropdown для выбора телефонного кода страны.

## Запуск проекта

1. Клонируйте репозиторий:

```bash
git clone <repo-url>
cd main-page
```

2. Откройте `index.html` в браузере.

Для более корректной локальной разработки можно поднять простой статический сервер, например:

```bash
npx serve .
```

или через VS Code Live Server.

## Что демонстрирует проект

- умение собирать landing page без frontend-фреймворков;
- аккуратную адаптивную верстку с кастомным UI;
- работу с интерактивными компонентами на чистом JavaScript;
- внимание к визуальной иерархии, CTA и UX-потоку страницы.
